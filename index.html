flexbox

flex-direction: modifies the main axis which, by default, has the value row, running horizontal from left to right. The opposite to this, row-reverse, runs horizontal from right to left. With the value column, the main axis now runs vertical from top to bottom. The opposite is column-reverse, running from bottom to top.

  • row
  • row-reverse
  • column
  • column-reverse

justify-content: aligns items along the main axis. Thus when flex-direction has the values row / row-reverse, items are adjusted horizontally. Likewise, when flex-direction has the values column / column-reverse, items are adjusted vertically.

  • flex-start
  • flex-end
  • space-between
  • space-around
  • space-evenly

align-item: aligns items along the cross axis. Thus when flex-direction has the values row / row-reverse, items are adjusted vertically. Likewise, when flex-direction has the values column / column-reverse, items are adjusted horizontally.

Note : align-content: appears to override align-items when there's the attribute flex-wrap: wrap.
